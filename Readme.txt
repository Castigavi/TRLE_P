######################################################################
#TROUBLESHOOTING:
######################################################################

Question: when I run the level, I get "DirectX error" message.
Answer: You need to run -setup again. Make sure your resolution isn't higher than 1920x1080. If you're still having issues, try using dgvoodoo2 wrapper.

Question: I get error "cann't load flep.dll".
Answer: You might need to install Microsoft Visual C++ 2010 Redistributable Package. Links:
https://titak.tombraiderhub.com/trng-plugins/vcredist_x86.exe
https://titak.tombraiderhub.com/trng-plugins/vcredist_x64.exe

Question: How can I get high resolution?
Answer: The updated version should already support HD resolutions. If the game still crashes though, try using LegacyD3DResolutionHack.Link: https://github.com/UCyborg/LegacyD3DResolutionHack
